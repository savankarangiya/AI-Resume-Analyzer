# 🚀 AI Career Copilot

AI-powered Resume Analyzer built with Streamlit and Groq LLM that helps job seekers improve their resumes through ATS scoring, Job Description matching, Missing Skills analysis, Career Recommendations, Resume Optimization, and Project Extraction.

## ✨ Features

* 📊 ATS Resume Score Analysis
* 💼 Resume vs Job Description Matching
* 🧩 Missing Skills Detection
* 🎯 Career Recommendations
* 🔍 Resume Issues Detection
* 🛠 AI Resume Optimization
* 📂 AI Project Extraction
* 📄 PDF Resume Upload Support
* ⬇️ Download Improved Resume in PDF & DOCX

## 🛠 Tech Stack

* Python
* Streamlit
* Groq API
* PyPDF
* Python-Docx
* ReportLab
* Regex (re)
* HTML/CSS

## 🚀 How to Run

### Clone Repository

```bash
git clone https://github.com/savankarangiya/AI-Career-Copilot.git
cd AI-Career-Copilot
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Add Environment Variable

Create a `.env` file:

```env
GROQ_API_KEY=your_api_key_here
```

### Run Application

```bash
streamlit run app.py
```
